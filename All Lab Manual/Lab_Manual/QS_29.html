Features of jQuery

1. DOM Manipulation: jQuery allows you to easily select and modify HTML elements with simple syntax.

2. Event Handling: You can attach event listeners to elements such as click, mouseenter, keypress, etc., effortlessly.

3. Animation Effects: jQuery provides built-in animation methods like fadeIn, fadeOut, slideUp, slideDown, and custom animations.

4. AJAX Support: jQuery simplifies AJAX calls and lets you dynamically load content without reloading the page.

5. Cross-Browser Compatibility: jQuery handles browser inconsistencies so your code works on all major browsers.
